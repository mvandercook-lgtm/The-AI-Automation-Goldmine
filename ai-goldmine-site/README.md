# The AI Automation Goldmine

> The definitive blog platform for entrepreneurs mastering AI tools, internet marketing, and business automation.

## Site Structure

```
ai-goldmine-site/
├── index.html              # Home page (Hero, Features, Blog, Spotlight, Newsletter)
├── css/
│   └── style.css           # Full design system & component styles
├── js/
│   └── main.js             # Animations, nav, counters, terminal
├── pages/
│   ├── about.html          # About page
│   └── resources.html      # Resources / Startup Toolkit
├── blog/
│   ├── index.html          # Blog archive with category filters
│   └── ai-content-engine.html  # Sample article
├── images/                 # (Add your images here)
└── .nojekyll               # Enables GitHub Pages with underscore paths
```

## Tech Stack
- Pure HTML5 / CSS3 / Vanilla JavaScript
- Google Fonts: Syne + Inter + Fira Code
- Zero dependencies — blazing fast load times
- GitHub Pages ready

## Deploy to GitHub Pages
1. Push this folder to your repo
2. Settings → Pages → Source: Deploy from a branch → `main` / `root`
3. Your site goes live at `https://[username].github.io/[repo-name]/`
